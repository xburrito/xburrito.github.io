Data Structures

Course Code: Csc 21200

Langauges used: C++

In this course, I have gone over the basics of data structures.

Linked Lists

Description: We covered the functionality of pointers and nodes in order to construct linked lists which could be dynamically resized unlike predefined arrays. On top of constructing a linked list, we gone over appending a node to the head, tail, or anywhere within the linked list.

Stacks

Description: Stacks follow last in, first out (LIFO) procedures. Think of it as a stack of pancakes. The freshest pancake just cooked is placed at the very top of the stack and would be the first one removed from the stack during consumption.

Queues

Description: The following topic discussed follows first in, first out (FIFO) procedures. The structure is a first come, first serve policy if we assume no other rules apply.

Priority Queues

Description: This topic is similar to stacks and queues. However, we add a in the variable: priority to establish importance of the nodes coming into the data structure. The sorting of priority may be numerical or simply labeled HIGH, MED, LOW priority which would indicate the importance of its completion.

Trees

Description: For this topic, we mainly stuck to general binary trees, the relationship between parent nodes and their children (parent nodes may have two, one, or no children nodes) as well as the traversing of nodes.

Graphs

Description: Unlike binary trees, graphs are structured with nodes connected by edges. Topics include traversing between nodes through the edges and algorithms used to optimize travel from starting node to destination i.e. Dijkstra

Image Processing

Course Code: Csc 47000

Langauges used: C++

This course involves using compression based algorithms alongside A.I. to modify image samples as well as reverse the process back to the original state or even aim to improve its original quality. The topics covered throughout the course:

  • Ability to implement point operations such as:
    • thresholding
    • quantization
    • gamma correction
    • histogram manipulation
    • halftoning
    • implement neighborhood operations such as:
      • blurring
      • sharpening
      • convolution
  • Knowledge of image reconstruction, re-sampling and antialiasing.
  • Knowledge of multipass transforms for efficient processing.
